B) CSS Files and Structure - top

There are 3 stylesheets used in this theme:

  1. css/reset.css
  2. css/jquery.scrollpane.css
  3. css/style.css
  1. reset.css is the reset stylesheet and is included in all pages.
  2. jquery.scrollpane.css is the stylesheet for the scrolling portion of the index.html and is only included on the index.html page.
  3. style.css is the main stylesheet and is included in all pages. It contains all the structural stylings for the website.

C) Fonts - top

D) JavaScript - top

This theme imports several Javascript files.

  1. js/jquery.easing-sooper.js
  2. js/jquery.js
  3. js/jquery.scrollpane.min
  4. js/jquery.kwicks-1.5.1.js
  5. js/jquery.mousewheel.js
  6. js/jquery.sooperfish.js
  7. js/modernizr-1.5.min.js
  1. easing-sooper is a jQuery plugin for easing transitions, optimised for the Sooperfish drop-down menu.
  2. jquery is a Javascript library containing lots of commonly used Javascript functions.
  3. scrollpane is a cross-browser jQuery plugin by Kelvin Luck which converts a browser's default scrollbars (on elements with a relevant overflow property) into an HTML structure which can be easily skinned with CSS.
  4. kwicks is a jQuery plugin for the image transitions.
  5. mousewheel is a jQuery plugin that adds cross-browser mouse wheel support.
  6. sooperfish is a jQuery plugin for the drop-down menu.
  7. modernizr is a Javascript library that enables HTML5 elements and feature detects.
